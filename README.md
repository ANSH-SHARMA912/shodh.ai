# shodh.ai
# Shodh Hiring — Policy Optimization for Financial Decision-Making

This repository contains an end-to-end pipeline for a loan approval decision-making task: EDA, preprocessing, supervised model, offline RL (CQL), and offline policy evaluation.

## Repo structure
- `notebooks/` — Jupyter notebooks (EDA, supervised training, RL training & evaluation, analysis).
- `src/` — reusable modules: data_utils.py, modeling.py, rl_agent.py, evaluation.py.
- `data/` — place your CSV here (not included if large). See `data/README.md`.
- `artifacts/` — saved preprocessor and models.
- `report/final_report.md` — 2–3 page final report (PDF available).
